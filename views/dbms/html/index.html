Dashboard

This database is designed using MySQL. The front-end UI uses the Bootstrap framework. Data is validated and captured by the client, then sent to the server where it is processed into SQL queries. These queries are then sent to the database, the results of which are returned to the server, forwarded to the client, and formatted.

There are currently two kinds of records: animals and employees.

Animal records contain:

  • Unique ID (primary key)
  • Name
  • Species
  • Type (ie. mammal, reptile)
  • Exhibit (ie. arctic, savannah)

Employee records contain:

  • Unique ID (primary key)
  • Name
  • Type (ie. handler, maintenance)
  • Subtype (ie. handler exhibit, maintenance department)
